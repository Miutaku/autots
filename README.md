## autots？

*automatic timeshift reservation system*の略です  
ニコニコ生放送の公式生放送のなかで見たい放送を自動でタイムシフト予約されます  
（検索する放送の範囲や検索条件は，適宜変更することができます）  

## できること

- 自動でタイムシフト予約したい公式生放送をタグで指定して登録
- 日付変更時，その日に放送される公式生放送から登録したタグが含まれる放送を自動でタイムシフト予約

## 使い方

### 必要なもの

- ruby
- redis

### gem

```
$ bundle install
```

### サーバの起動

```
$ thin start -d
```
